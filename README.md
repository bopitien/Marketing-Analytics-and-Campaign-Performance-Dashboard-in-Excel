# Marketing-Analytics-and-Campaign-Performance-Dashboard-in-Excel


This project presents a complete Excel based marketing analytics solution that tracks campaign performance, engagement trends and return on investment. It is designed to help marketing teams understand what drives impressions, clicks, conversions and ROI while also providing clear insights and priorities that support decision making.

The workbook contains two interactive dashboards and one insights page. Together they show how Excel can be used to organize large scale marketing performance data and communicate results in a simple visual manner.

---

##  Project Objective

To build a comprehensive Excel dashboard system that monitors campaign engagement and performance across multiple channels, locations and audience groups. The goal is to give stakeholders a clear understanding of what is working, where value is coming from and which actions should be prioritized.

---

##  Dataset Summary

The dataset includes:

- Monthly campaign records for the year 2021  
- Impressions and clicks  
- Conversion rate and acquisition cost  
- ROI and spend  
- Campaign type  
- Channel used  
- Location  
- Target audience segments  

This dataset allows for channel analysis, ROI comparison, audience targeting insights and time trend evaluation.

---

##  Tools and Techniques

- Microsoft Excel for dashboard development  
- Pivot tables  
- Pivot charts  
- Slicers for date filtering  
- Form controls and card style KPI blocks  
- Line charts for trend analysis  
- Bar charts for comparative analysis  
- Donut charts for audience segmentation

#  Insights, Recommendations and Priorities

<img src="insights_page_placeholder.png" width="900"/>

This page summarizes findings from the two dashboards in a structured easy to understand format.

---

##  Insights

- Impressions and clicks both increased by approximately nine percent versus previous month  
- Conversion rate is strong at eight point zero one percent  
- Acquisition cost reduced by four percent  
- ROI increased significantly by one hundred and nine percent  
- Miami and Los Angeles show the highest ROI performance  
- Google Ads drives the highest impressions  
- Email channel delivers the highest ROI  
- Influencer campaigns generate the best ROI among campaign types  
- Men aged eighteen to twenty four are the highest impression segment  

---

## Recommendations

- Increase budget allocation to Miami and Los Angeles  
- Strengthen email marketing through improved automation and content personalization  
- Expand influencer campaigns and track conversion impact  
- Continue using Google Ads for visibility but refine targeting  
- Focus campaign targeting on men eighteen to twenty four  

---

##  Priorities

- Shift part of campaign budget to top performing locations and channels  
- Launch targeted email and influencer campaigns for men eighteen to twenty four  
- Run A B tests on landing pages and email content  
- Review website conversion process and reduce friction points  
- Track acquisition cost and ROI weekly and adjust spend if needed

##  How to Use

1. Open the Excel file in the Dashboard folder  
2. Navigate between the dashboard pages using the icons in the left sidebar  
3. Use the month slicers to filter data for specific periods  
4. Review the insights page for clear next steps and business actions  
5. Apply this dashboard for campaign optimization, audience targeting and budget allocation
